# Ang Xu Academic Website

This is a simple static website for `angxu1.github.io`. It uses a classic academic layout inspired by jemdoc-style personal pages: left navigation, serif typography, blue headings, and separate pages for publications and experiences.

## Edit the Site

- Update `index.html` with bio, contact information, education, and research interests.
- Update `publications.html` with publications and preprints.
- Update `experiences.html` with internships, teaching, service, and honors.
- Replace `cv.pdf` whenever you want to update the downloadable CV.
- Replace `profile.jpg` whenever you want to update the profile photo.

## Publish to GitHub Pages

1. Create a public GitHub repository named `angxu1.github.io`.
2. Upload `index.html`, `publications.html`, `experiences.html`, `cv.html`, `styles.css`, `.nojekyll`, `cv.pdf`, and `profile.jpg`.
3. In the repository, go to `Settings` -> `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose the `main` branch and `/root`, then save.
6. Your site should appear at `https://angxu1.github.io/` after GitHub finishes deploying.

## Local Preview

Open `index.html` in a browser, or run any static server from this folder. Because this site has no build step, what you see locally is what GitHub Pages will serve.
